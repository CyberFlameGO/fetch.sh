## How to Use:
Clone the repository

> **$**&ensp; `git clone -b master --single-branch https://github.com/wick3dr0se/fetch.sh`

Change directory into the folder

> **$**&ensp; `cd fetch.sh`

Make script executable

> **$**&ensp; `chmod a+rx fetch.sh`

Execute script

> **$**&ensp; `./fetch.sh`
